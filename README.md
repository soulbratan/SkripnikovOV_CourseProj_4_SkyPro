# 📧 Система управления рассылками (SkripnikovOV SkyPro, Курсовая работа №4)

Django-приложение для управления email-рассылками с возможностью создания, отправки и отслеживания рассылок для различных клиентов.

## 🚀 Возможности

### 📊 Основной функционал
- **Управление клиентами** - CRUD операции с получателями рассылок
- **Создание сообщений** - шаблоны писем с темой и телом
- **Управление рассылками** - планирование и отправка email-рассылок
- **Отслеживание попыток** - мониторинг успешных и неудачных отправок
- **Статистика и отчеты** - детальная аналитика по рассылкам

### 👥 Пользовательская система
- **Регистрация и аутентификация** через email
- **Разделение прав доступа** (пользователи и менеджеры)
- **Восстановление пароля** по email
- **Личные кабинеты** для каждого пользователя

### ⚙️ Дополнительные возможности
- **Относительное время** для удобного планирования рассылок
- **Кеширование** для повышения производительности
- **REST-like API** для интеграций
- **Адаптивный интерфейс** на Bootstrap 5

## 🛠️ Технологический стек

- **Backend**: Django 4.2.7
- **Database**: PostgreSQL
- **Frontend**: Bootstrap 5, Crispy Forms
- **Cache**: Redis
- **Email**: SMTP (Gmail/Yandex/другие провайдеры)

## 📦 Установка и запуск

1. **Клонируем репозиторий:**
```
git clone https://github.com/soulbratan/SkripnikovOV_CourseProj_4_SkyPro.git
```
2. **Добавляем интерпретатор poetry:**

3. **Устанавливаем зависимости:**
```
poetry update
```

4. **Настраиваем базу данных и почтовую рассылку в переменных окружения**
```
Добавляем .env файл и заполняем по примеру .env.sample
```

5. **Производим инициализацию базы данных**
```
- python manage.py migrate

- python manage.py create_groups

- python manage.py createsuperuser
```

6. **Запускаем сервер**
```
python manage.py runserver
```

## 👤 Роли и права доступа

### 🔧 Менеджер
- Просмотр всех рассылок и клиентов
- Блокировка пользователей
- Отключение рассылок
- Доступ к панели управления

### 📧 Пользователь
- Создание и управление своими рассылками
- Просмотр статистики по своим рассылкам
- Управление своими клиентами и сообщениями


### 🎯 Использование
1. **Регистрация и вход**
   - Перейдите на главную страницу
   - Зарегистрируйтесь или войдите в систему
   - Настройте профиль пользователя

2. **Создание клиентов**
    - Перейдите в раздел "Клиенты"
    - Добавьте получателей рассылок с email и комментариями

3. **Создание сообщений**
    - В разделе "Сообщения" создайте шаблоны писем
    - Укажите тему и тело письма

4. **Настройка рассылки**
   - В разделе "Рассылки" создайте новую рассылку
   - Выберите время начала и длительность
   - Укажите сообщение и получателей
   - Запустите рассылку

5. **Мониторинг**
    - Отслеживайте попытки отправки в разделе "Попытки"
    - Анализируйте статистику в соответствующем разделе
